  • 讲解：

    • 通俗理解：你现在看到的这个文件，就是未来网页版说明书的首页 (Home Page)。Sphinx 会读取这个文件，把它变成漂亮的 HTML 网页。

目标：读懂正文段落，知道 Megatron Core 是什么。

  • 讲解：
    • 核心定义：Megatron Core 是一个 Python 库（工具箱）。

目标：理解 .. toctree:: 指令，知道文档分成了哪两部分。

    .. toctree:: :caption: API Guide api-guide/index `` * **讲解**： *toctree的意思是 **"Table of Contents Tree"（目录树）**。这部分代码告诉 Sphinx：“请在这里生成两个导航菜单”。 * **第一部分：用户指南 (User Guide)** * 这是给人类看的教程。 * 它会链接到user-guide/index这个文件。 * **你应该先看这个**，因为它会教你基本概念和流程。 * **第二部分：API 指南 (API Guide)** * 这是给程序员查阅的“字典”。 * 它会链接到api-guide/index` 这个文件。 * 这里面列出了具体的函数名、参数和代码细节。
