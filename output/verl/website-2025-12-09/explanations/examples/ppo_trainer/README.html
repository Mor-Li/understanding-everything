3. 给我一个高层的认知 (High-Level Insight)

要读懂这个文件夹，你只需要记住一个公式：

一个脚本 = 一套配置 (Config)

你不需要去读懂里面的 Python 逻辑，你只需要像点菜一样看懂这些脚本在配什么：

  1. 主角 (Actor)：你想训练谁？(DeepSeek? Qwen?) -> 去改 model.path
  2. 教材 (Data)：你想让它学什么？(数学? 写代码?) -> 去改 data.train_files
  3. 场地 (Hardware)：你家里有几张显卡？(8卡? 4卡?) -> 去改 n_gpus_per_node
  4. 助教 (Rollout)：你想让它答题快一点吗？ -> 确认 rollout.name=vllm
